## Descripción

Aplicación web full-stack que permite gestionar una galería de superhéroes. Los usuarios pueden registrarse, iniciar sesión y explorar información sobre diferentes superhéroes.

El proyecto está dividido en dos partes:

- **Backend**: API REST desarrollada con Node.js y Express, utilizando Sequelize para la persistencia de datos
- **Frontend**: Interfaz de usuario construida con React y Vite

## Tecnologías utilizadas

### Backend

- Node.js
- Express
- PHP my Admin
- JWT para autenticación

### Frontend

- React
- Vite
- React Router

## Instalación

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Características

- Sistema de autenticación con registro e inicio de sesión
- Visualización de galería de superhéroes
- Rutas protegidas para usuarios autenticados
- API RESTful para gestión de datos
